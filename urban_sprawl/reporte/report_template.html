Metodología

Para la detección de expansión urbana se utlizó el enfoque Detención de cambio de coberturras usando bandas de probabilidad, basado en el conjunto de datos Dynamic World de Google Earth Engine. Este conjunto de datos proporciona, para cada píxel y fecha, las probabilidades de pertenecer a diferentes clases de cobertura, entre ellas la clase Área construida. Se generaron dos composiciones anuales promedio de dicha clase: una correspondiente al periodo “antes” (por ejemplo, julio de 2025) y otra al periodo “después” (por ejemplo, agostos de 2025). Estas composiciones reflejan la probabilidad media de urbanización en cada píxel durante ambos periodos. A partir de estas imágenes, se aplicó un umbral probabilístico para identificar las zonas que pasaron de ser no urbanas a urbanas: píxeles con una probabilidad de urbanización inferior a 0.2 en el periodo anterior y superior a 0.5 en el periodo posterior se clasificaron como áreas de nueva expansión urbana. De este modo, el resultado obtenido corresponde a una capa binaria que indica la ocurrencia de cambio de uso del suelo hacia cobertura urbana.

Posteriormente, el resultado de expansión urbana se intersectó con tres capas ambientales y normativas clave:

  • Situación ambientalmente conflictiva: identifica zonas de alta presión antrópica o conflictos de uso del suelo.
  • Reserva Forestal Protectora Bosque Oriental de Bogotá: establecida mediante el régimen de la Reserva de Cerros Orientales, donde la urbanización está sujeta a fuertes restricciones.
  • Estructura Ecológica Principal (EEP): define las áreas estratégicas para la conectividad ecológica y la provisión de servicios ecosistémicos.

Estas intersecciones entre los nuevos procesos de expansión urbana y los espacios ambientalmente sensibles, permiten diferenciar preliminarmente las zonas en donde la construcción es permitida y las zonas en donde tiene restricciones

Para más información, consulte: Introducción a Dynamic Wolrd (parte 3).
